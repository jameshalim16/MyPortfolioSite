This is my styleguide for my portfolio project. This styleguide focuses on typography, hierachy, colour scheme and ethos.

Ethos

The ethos for my portfolio would be in the visual elements and feel of photography. Specifically when old photos would be processed in a red room. Those photos would be processed and hung for the next few hours. My goal for the overall visual elements of my portfolio is to have similar elements of photography. As the user scrolls and interacts with my portfolio, they will come across various projects and visual elements that I hope have the photography element feeling.

the typography

The typography uses Poppins font for the header and Inter font for the body paragraph.

the grid

The grid structure is composed of one column grid, a two column grid, a three column grid, and a four column grid. This grid is very flexible and can be changed depending on the content of any website. Although the grids can be used in a variety of ways, together, they help keep the overall visual of the website consistent.

the colours

The colours primary used in my portfolio will be black throughout the website. Essentially a dark mode website. I will also use white for fonts, and gradients for some buttons and headers.

Primary colour. #FFFFFF
All text. #000000

the image

The imagery used in the portfolio will be mainly my own personal project works. That would be screenshots of figma projects, final renders of project banners, as well as other elements that I've created for my portfolio.
